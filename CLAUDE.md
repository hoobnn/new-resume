# Commit Convention

This project uses **gitmoji + conventional commits**, enforced by husky + commitlint.

Format: `<emoji> <type>(<scope>): <description>`

Examples:

- `宖脂 feat: init project`
- `宖菅 fix(auth): handle token expiry`
- `笨ｨ feat(ui): add dark mode`

Use `gitmoji -c` for interactive commits.
